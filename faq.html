Software & Hardware

What are the software requirements?

DoveTailTools spreadsheets require Microsoft Excel to be installed on the workstation.

DoveTailTools spreadsheets have been verified to operate correctly with Microsoft Windows 11 Pro and the Excel version included in Microsoft 365. The spreadsheets include formulas that were not available prior to Excel 2019.

What hardware is recommended?

An Intel Core i3 or AMD Ryzen 3 processor (or better), and a minimum of 16 GB of RAM, are recommended.

Licenses

For the volume discount, does each bundle count as a single license or as multiple licenses?

Each bundle counts as multiple licenses toward the volume discount. Each HVAC bundle counts as two licenses, and each Plumbing bundle counts as four licenses.

Are volume discounts determined by the total number of licenses that a company has, or are volume discounts location‑specific?

Volume discounts are location‑specific. For license agreement purposes, the location is the address on the invoice.

Can a license be installed on multiple workstations for the same user?

No. Each license applies to a single workstation.

File Format

What is the format of the files?

DoveTailTools spreadsheets are delivered as encrypted Windows executable (.exe) files. The files are created using XLS Padlock, a commercial tool designed specifically for creating encrypted executable files from Excel spreadsheets. Though the files are Excel spreadsheets, they are opened similar to other executable Windows programs. They are not opened directly in Excel.

Are the executable files safe to run?

Yes. DoveTailTools executable files are digitally signed using a trusted code signing certificate issued by DigiCert, a widely recognized certificate authority, that helps prevent security warnings and administrative blocks that are common with unsigned executable files. The digital signatures verify that the files were created by DoveTailTools and have not been tampered with or altered.

File Naming & Copying

Can the executable files be renamed?

Yes. The executable files can be renamed for ease of identification, as long as the .exe extension is maintained. Renaming the executable files does not affect the licensing or the functionality.

Can the executable files be copied?

Yes. The executable files may be copied as needed.

File Location

Can the executable files be located on a server?

Yes. The executable files can be located on a shared server. Any workstation that has a license for the spreadsheet can run the files directly from the server. Workstations without a license will not be able to run the files, even if they can access them.

File Linking

Can the executable files be linked to other spreadsheets or other applications?

No. The transformation of the spreadsheets into encrypted executable files removes Excel's external link capability.

Can values in the executable files be copied and pasted to other documents?

Yes. Values can be copied and pasted to other documents, and in some cases the formatting may also be copied. However copying from cells with formulas will copy the values in the cells only; copying from cells will not copy the formulas in the cells.

Can values from other files be copied and pasted to the executable files?

Yes. Values can be copied and pasted from other documents, however they can only be pasted to cells designated for user input. Formulas cannot be pasted from other documents; formulas pasted from other documents are converted to their resulting values.
