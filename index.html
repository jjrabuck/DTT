Why Spreadsheets Excel

A few advantages of DoveTailTools spreadsheets compared to …

… programs

No “Running”

Unlike most programs, spreadsheet values update instantly with every press of Enter.

No Squinting

High resolution monitors and fixed pixel fonts can strain eyes. Spreadsheets are fully zoomable and pannable.

No Missing References

DoveTailTools spreadsheets are self contained. There are no external reference files and no external links.

Customizable Views

Spreadsheets allow multiple views resizable views, whereas program interfaces are nonmutable.

… modeling programs

Specialization

Modeling programs are Swiss army knives. Spreadsheets are optimized for a specific purpose, with features designed for that purpose that modeling programs lack.

Navigation

DoveTailTools spreadsheets have a simple intuitive tabbed interface that is much easier to navigate than the labyrinth of menus of modeling programs.

Productivity

Modeling programs require systems to be 'connected' before calculations can be run. Connecting systems is tedious and budget intensive.

Resource Utilization

Staff responsible for modeling may not be the same staff responsible for calculations. Spreadsheets let the right people focus on the right tasks.

… your custom spreadsheets

Features

DoveTailTools spreadsheets include features such as profiles, wizards, and templates that dramatically speed up both data entry and adjusting sizes to optimize systems.

Instant Discrepancy Feedback

Renaming segments and zones can create reference discrepancies. DoveTailTools spreadsheets highlight such discrepancies instantly via text color feedback.

Complete Systems

DoveTailTools spreadsheets are configured for designing complete systems, while still allowing experienced users to condense input when the critical path is obvious.

User Manuals

Each spreadsheet includes a comprehensive user manual for clarity and confidence.
